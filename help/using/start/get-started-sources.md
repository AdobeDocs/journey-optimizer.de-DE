# Erste Schritte mit Quell-Connectoren {#sources-gs}

Adobe Journey Optimizer ermöglicht die Aufnahme von Daten aus externen Quellen und bietet spezielle Platform-Services, mittels derer Sie eingehende Daten strukturieren, beschriften und erweitern können. Daten können aus verschiedensten Quellen aufgenommen werden, darunter etwa Adobe-Anwendungen, Cloud-basierte Datenspeicher und Datenbanken.

![](assets/sources-home.png)

In diesem Video erfahren Sie mehr über Quell-Connectoren und deren Konfiguration in Journey Optimizer:

>[!VIDEO](https://video.tv.adobe.com/v/335919?quality=12)

Weitere Informationen zum Aufnehmen von Daten aus externen Quellen mit Quell-Connectoren finden Sie in der [Dokumentation zu Adobe Experience Platform](https://experienceleague.adobe.com/docs/experience-platform/sources/home.html?lang=de){target="_blank"}.
