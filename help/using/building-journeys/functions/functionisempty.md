# isEmpty {#isEmpty}

Gibt „true“ zurück, wenn die Zeichenfolge im Parameter keine Zeichen enthält.

## Kategorie

Zeichenfolge

## Funktionssyntax

`isEmpty(<parameters>)`

## Parameter

* string

## Signatur und zurückgegebener Typ

`isEmpty(<string>)`

Gibt einen booleschen Wert zurück.

## Beispiel

`isEmpty("")`

Gibt „true“ zurück.

`isEmpty("Hello World")`

Gibt „false“ zurück.
