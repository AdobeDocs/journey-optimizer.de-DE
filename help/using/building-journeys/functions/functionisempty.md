# isEmpty {#isEmpty}

Gibt &quot;true&quot;zurück, wenn die Zeichenfolge im Parameter kein Zeichen enthält.

## Kategorie

Zeichenfolge

## Funktionssyntax

`isEmpty(<parameters>)`

## Parameter

* Zeichenfolge

## Signatur und zurückgegebener Typ

`isEmpty(<string>)`

Gibt einen booleschen Wert zurück.

## Beispiel

`isEmpty("")`

Gibt &quot;true&quot;zurück.

`isEmpty("Hello World")`

Gibt false zurück.
