# indexOf {#indexOf}

Gibt die Position (im ersten Argument) des ersten Vorkommens des zweiten Parameters zurück. Gibt -1 zurück, wenn keine Übereinstimmung vorliegt.

## Kategorie

Zeichenfolge

## Funktionssyntax

`indexOf(<parameters>)`

## Parameter

| Parameter | Typ |
|-----------|------------------|
| string | Zeichenfolge |
| angegebener Wert | Zeichenfolge |

## Signatur und zurückgegebener Typ

`indexOf(<string>,<string>)`

Gibt eine Ganzzahl zurück.

## Beispiel

`indexOf("Hello", "l")`

Gibt 2 zurück.

Erklärung:

In „Hello“ befindet sich das erste Auftreten von „l“ an Position 2.
