# lower {#lower}

Gibt eine Version des Parameters in Kleinbuchstaben zurück.

## Kategorie

Zeichenfolge

## Funktionssyntax

`lower(<parameter>)`

## Parameter

* Zeichenfolge

## Signatur und zurückgegebener Typ

`lower(<string>)`

Gibt eine Zeichenfolge zurück.

## Beispiel

`lower("A")`

Gibt &quot;a&quot;zurück.
