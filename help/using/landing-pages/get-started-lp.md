# Erste Schritte mit Landingpages {#get-started-lp}

Eine Landingpage ist eine eigenständige Web-Seite, auf die ein Benutzer geleitet wird, nachdem er auf eine E-Mail, eine Website, eine Anzeige oder einen anderen digitalen Ort geklickt hat.

[!DNL Journey Optimizer] können Sie Landingpages erstellen und gestalten, um Ihre Benutzer zu Online-Formularen zu leiten, über die sie sich für den Erhalt Ihrer Nachrichten oder eines bestimmten Services wie eines Newsletters anmelden oder abmelden können.

➡️ [Informationen zur Konfiguration von Abonnements und zur Erstellung von Landingpages finden Sie in diesem Video](#video)

## Verwendung von Landingpages {#when-to-use}

Verwenden Sie Landingpages, wenn Sie Folgendes tun möchten:

* Kunden die Möglichkeit **, sich gegen Marketing** Nachrichten, einen bestimmten Service oder Newsletter über einen Link in einer E-Mail oder Kampagne zu entscheiden
* **Einverständnis einholen** bevor Nachrichten gesendet werden, und die Aktion mit einer automatisierten E-Mail bestätigen
* Benutzer zu einem **Web-Formular umleiten** ohne eine externe Seite außerhalb von [!DNL Journey Optimizer] zu erstellen

## Bevor Sie beginnen {#prerequisites}

Führen Sie vor dem Erstellen einer Landingpage die folgenden Schritte aus:

1. [**Subdomain konfigurieren**](lp-subdomains.md) - Richten Sie eine Subdomain ein, die dem Hosten Ihrer Landingpages dient.
1. [**Landingpage-Voreinstellung erstellen**](lp-presets.md#lp-create-preset) - Eine Voreinstellung definiert die Subdomain und andere Einstellungen, die auf Ihre Landingpages angewendet werden.
1. [**Abonnement-Liste erstellen**](subscription-list.md) (für Abonnement-Anwendungsfälle) - Erforderlich, wenn Kundinnen und Kunden einen bestimmten Dienst abonnieren oder abmelden sollen.

## Funktionsweise {#how-it-works}

Das Erstellen und Bereitstellen einer Landingpage folgt dieser Reihenfolge:

1. [**Landingpage erstellen und konfigurieren**](create-lp.md) - Wählen Sie eine Voreinstellung aus, richten Sie die Primärseite ein und fügen Sie alle erforderlichen Unterseiten hinzu.
1. [**Seite gestalten**](design-lp.md) - Seiteninhalt und Formular mit dem Drag-and-Drop-Editor von [!DNL Journey Optimizer] erstellen.
1. [**Testen und Veröffentlichen**](manage-lp.md) - Vorschau der Seite, Testen des Formularverhaltens und anschließendes Veröffentlichen, um sie live zu schalten.
1. [**Link in einer Nachricht oder einer Journey**](../email/message-tracking.md#insert-links) - Fügen Sie die Landingpage-URL zu einer E-Mail-, Kampagnen- oder Journey-Aktion hinzu, damit Kunden sie erreichen können.

## Wichtigste Funktionen {#capabilities}

* Nutzen Sie [!DNL Journey Optimizer]Funktionen zur Inhaltserstellung zur mühelosen Gestaltung von **responsiven Landingpages**.
* Richten Sie **Opt-in- und Opt-out-Vorgänge** schnell und nahtlos ein, einschließlich Abonnement-Listen, über die Benutzer einen bestimmten Service aktivieren oder deaktivieren können. [Weitere Informationen](lp-use-cases.md#subscription-to-a-service)
* Senden Sie beim Opt-in oder Opt-out eine **Bestätigungs-E-Mail**. [Weitere Informationen](lp-use-cases.md#send-confirmation-email)
