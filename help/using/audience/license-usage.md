# Lizenznutzungs-Dashboard {#license-usage}

Die [!DNL Adobe Journey Optimizer] [Benutzeroberfläche](../start/user-interface.md) bietet ein Dashboard, das wichtige Informationen zur Lizenznutzung Ihrer Organisation anzeigt, wie sie bei einem täglichen Snapshot erfasst werden.

Um auf das Dashboard zuzugreifen, gehen Sie zu **[!UICONTROL Administration]** > **[!UICONTROL Lizenznutzung]**. Dadurch wird die Registerkarte **[!UICONTROL Übersicht]** geöffnet, auf der das Dashboard angezeigt wird.

![Überblick über das Lizenznutzungs-Dashboard](assets/license-usage-dashboard.png)

>[!NOTE]
>
>Um das Dashboard anzuzeigen, benötigen Sie die [Anzeigen des Lizenznutzungs-Dashboards](https://experienceleague.adobe.com/docs/experience-platform/dashboards/permissions.html?lang=de#available-permissions){target="_blank"} Berechtigung.

[!DNL Adobe Journey Optimizer] können Sie im Dashboard die Anzahl der (**Profile)**. Ein kontaktierbares Profil ist ein Datensatz mit Informationen, die eine Person repräsentieren und im Profil-Service gespeichert werden. Diese Einträge sind Profile, mit denen Sie in den letzten 12 Monaten mithilfe der Authoring-, Entscheidungs-, Versand-, Experimentierungs- oder Orchestrierungsfunktionen von Journey Optimizer interagiert haben.
